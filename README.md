# PropertyLedger

A clean, modular rental management application for property owners and managers.

## Features

- **Dashboard**: High-level overview of income, expenses, and net operating income.
- **Properties & Units**: Manage your real estate portfolio, including lease terms and tenant shares.
- **Rent Ledger**: Track monthly payments, auto-apply late fees, and manage utility billing.
- **Transactions**: Full history of income and expenses with categorical filtering.
- **Cash Flow**: Monthly performance visualization.
- **Violations & Inspections**: Track code violations, annual inspections, and associated fines.
- **Tax Report**: Automatic Schedule E summary for easy tax preparation.

## Project Structure

- `index.html`: The main structure and entry point.
- `style.css`: Modern, dark-themed styling with premium aesthetics.
- `app.js`: Application logic and state management.

## Getting Started

Simply open `index.html` in any modern web browser to start managing your properties. Data is currently managed in-memory for this session.

## Design

Built with a focus on ease of use and professional aesthetics:
- **Typography**: DM Sans and DM Serif Display.
- **Color Palette**: Sophisticated dark theme with curated accent colors.
- **Responsive**: Designed to work across different screen sizes.
